The Automatic Speech Recognition for Spanish Language Project

http://www.speech.cs.cmu.edu/sphinx/models/hub4spanish_itesm/

The purpose of this site is to give the reader the main files and source code to straightforward construct an automatic speech recognition system for spanish.

    * Acoustic Models for Spanish (1g,2g, 4g, 8g, 16g,32g)
    * Model Definition file for Spanish (H4.2500.mdef)
    * Language Models for Spanish (H4.arpa.Z.DMP)

    * Source code

          o Please check the CMUSphinx site.

    * Files for Training

         1. Dictionary (h4.dict)
         2. List of phonemes (list_of_phonemes.unique.basic.txt) 
         3. Control file (train.ctl_tst25pct_c)
         4. Label file (train.corpus_tst25pct_c)
         5. Filler file (filler.dict)
